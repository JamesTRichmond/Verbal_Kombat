# Today's execution goals

## Codex owns

1. **Playable vertical slice** — one complete round from opponent claim through score, damage, verdict, and replay.
2. **Trustworthy rules** — scoring and combat are pure, deterministic modules with automated win/loss and discipline checks.
3. **Maintainable content** — prompts and opponent data live outside the interface so later bouts do not require UI rewrites.
4. **Release confidence** — run syntax/tests, exercise representative arguments, and document remaining Release 1 gates.
5. **Integration** — push the verified foundation, review Copilot pull requests, and prevent scope or file overlap.

## Copilot owns

1. Contributor onboarding and architecture/decision documentation.
2. GitHub issue forms, pull-request template, and CODEOWNERS.

## Definition of real progress today

- The first bout is playable and reproducible.
- Core scoring and combat logic pass automated checks.
- Copilot has independent, mergeable background work.
- The repository has a clear path from prototype to playtest.
