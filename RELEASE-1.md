# Release 1 acceptance checklist

- [x] Loads in a modern browser without a build step.
- [x] Presents one prompt and opponent claim.
- [x] Accepts and scores a counterargument.
- [x] Separately shows claim, support, rebuttal, and discipline.
- [x] Applies damage and shows a round win or loss.
- [x] Supports replay, keyboard use, and narrow screens.
- [x] Includes automated scoring checks.
- [ ] Complete a five-person playtest and fix completion blockers.
- [ ] Publish at a shareable URL.
- [ ] Tag the tested commit as `v1.0.0`.

Voice, AI judging, multiplayer, accounts, progression, and additional fighters are explicitly post–Release 1.
